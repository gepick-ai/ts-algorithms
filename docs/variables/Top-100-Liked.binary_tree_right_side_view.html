    199.二叉树的右视图

    给定一个二叉树的根节点 root，返回其 右视图 。

    二叉树的 右视图 是从树的右侧看过去，按从上到下的顺序依次得到的节点值。

    示例 1：

    输入： root = [1,2,3,null,5,null,4]

    输出： [1,3,4]

    示例 2：

    输入： root = [1,null,3]

    输出： [1,3]

    示例 3：

    输入： root = []

    输出： []

    提示：

    • 二叉树的节点数在范围 [0, 100] 内
    • -100 <= Node.val <= 100
