# [51.N皇后](https://leetcode.cn/problems/n-queens/description)

按照国际象棋的规则，皇后可以攻击同一行、同一列及同一条斜线上的棋子。

给你一个整数 `n`，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个不同的 n 皇后问题的棋子放置方案，该方案中 `'Q'` 和 `'.'` 分别代表了皇后和空位。

---

## 示例 1：

**输入：** `n = 4`

**输出：**
```
[
  [".Q..","...Q","Q...","..Q."],
  ["..Q.","Q...","...Q",".Q.."]
]
```

---

## 示例 2：

**输入：** `n = 1`

**输出：**
```
[["Q"]]
```

---

## 提示：

- `1 <= n <= 9`
