# [42.接雨水](https://leetcode.cn/problems/trapping-rain-water/description)

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

---

## 示例 1：

![示例1](../images/42.接雨水.png)

**输入：** `height = [0,1,0,2,1,0,1,3,2,1,2,1]`

**输出：** `6`

**解释：** 如下图所示，6 单位雨水被接收。

---

## 示例 2：

**输入：** `height = [4,2,0,3,2,5]`

**输出：** `9`

---

## 提示：

- `n == height.length`
- `1 <= n <= 2 * 10^4`
- `0 <= height[i] <= 10^5`
