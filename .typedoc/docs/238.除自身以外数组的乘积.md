# [238.除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/description)

给你一个整数数组 `nums`，请你返回一个数组 `answer`，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积。

题目要求时间复杂度为 O(n)，且不能使用除法。

---

## 示例 1：

**输入：** `nums = [1,2,3,4]`  
**输出：** `[24,12,8,6]`

---

## 示例 2：

**输入：** `nums = [-1,1,0,-3,3]`  
**输出：** `[0,0,9,0,0]`

---

## 提示：

- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`
- 保证数组中任意元素的乘积都在 32 位整数范围内
